# Notices

AgentMux Windows is an independent Windows implementation.

It is inspired by the workflow category popularized by cmux, including terminal workspaces, panes, sidebar status, agent notifications, and programmable local control. It is not affiliated with cmux, Manaflow, or the macOS cmux project.

No source code, assets, screenshots, branding, or documentation from `manaflow-ai/cmux` are included in this repository.

The public repositories below were used as references for product behavior and Windows feasibility:

- https://github.com/manaflow-ai/cmux
- https://github.com/mkurman/cmux-windows
- https://github.com/amirlehmam/wmux
